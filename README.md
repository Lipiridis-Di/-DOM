# Tasks List

Небольшое учебное приложение для управления списком задач на чистом JavaScript.

[Демо](https://lipiridis-di.github.io/-DOM/) • [Репозиторий](https://github.com/Lipiridis-Di/-DOM)

## Что умеет приложение

- отображает стартовый список задач;
- добавляет новые задачи через форму;
- проверяет поле на пустое значение;
- не позволяет создавать дубликаты задач;
- показывает сообщение об ошибке под формой;
- удаляет задачи через модальное окно подтверждения;
- переключает тему со светлой на тёмную по нажатию клавиши `Tab`.

## Запуск проекта

1. Склонировать репозиторий:

```bash
git clone https://github.com/Lipiridis-Di/-DOM.git
```

2. Перейти в папку проекта:

```bash
cd -DOM
```

3. Открыть файл `index.html` в браузере.

## Как пользоваться

- Введите название задачи в поле и нажмите `Создать`.
- Если поле пустое, приложение покажет ошибку.
- Если задача с таким названием уже существует, появится предупреждение.
- Чтобы удалить задачу, нажмите `Удалить` и подтвердите действие в модальном окне.
- Чтобы переключить тему, нажмите клавишу `Tab`.

## Особенности реализации

- Для удаления задач используется делегирование событий.
- Ошибки валидации создаются динамически в DOM через `span.error-message-block`.
- Переключение темы реализовано через inline-стили с помощью свойства `style`.
